API Reference
CREATE
Nova.new()
Nova.new(title, options?) ‚Üí Window
Creates a new UI window. Options: theme, position, size, draggable, dpi.
CREATE
:AddTab()
Window:AddTab(name) ‚Üí Tab
Adds a navigation tab to the window. Returns a Tab object for adding sections.
CREATE
:AddModule()
Tab:AddModule(title, desc?) ‚Üí Module
Creates a glassmorphic module card with optional description and power toggle.
ADD
:AddToggle()
Module:AddToggle(name, default?) ‚Üí Toggle
iOS-style pill switch. Returns a Toggle with .Value and :OnChanged(cb) event.
ADD
:AddSlider()
Module:AddSlider(name, min, max, default?) ‚Üí Slider
Glowing rounded slider. Supports suffix, decimals, and live value display.
ADD
:AddButton()
Module:AddButton(icon, label?) ‚Üí Button
Circular icon button. Supports active states: default, blue, green, red, amber.
ADD
:AddColorPicker()
Module:AddColorPicker(name, default?) ‚Üí ColorPicker
Color swatch row. Returns selected Color3. Supports custom palette arrays.
EVENT
:OnChanged()
Component:OnChanged(callback) ‚Üí void
Fires whenever the component value changes. Passes new value as first argument.
SET
:SetValue()
Component:SetValue(value) ‚Üí void
Programmatically update any component. Triggers OnChanged if value differs.
SET
:SetDPI()
Window:SetDPI(scale) ‚Üí void
Sets UI scale (0.5‚Äì3.0). Use with UserInputService to auto-detect mobile DPI.
SET
:Hide() / :Show()
Window:Hide() / Window:Show() ‚Üí void
Collapses to a minimized tab on screen edge. Tapping restores. Keyboard bindable.
ADD
:Notify()
Nova:Notify(title, msg, type?) ‚Üí void
Toast notification. Types: info, success, warn, error. Auto-dismisses after 4s.

-- // NovaUI v2 ‚Äî Full Example local Nova = loadstring(game:HttpGet("rbxassetid://NOVA_ID"))() -- Create window with DPI auto-detection local win = Nova.new("My Script", { theme = "glass", position = UDim2.fromScale(0.5, 0.5), dpi = Nova.AutoDPI(), -- mobile auto-scale draggable = true, hideKey = Enum.KeyCode.RightControl, }) -- Add tab & module local tab = win:AddTab("Main") local espMod = tab:AddModule("Player ESP", "Highlight players") -- Toggle local espToggle = espMod:AddToggle("Enable ESP", false) espToggle:OnChanged(function(v) ESP.SetEnabled(v) end) -- Slider with live callback local range = espMod:AddSlider("Range", 50, 1000, 500) range:OnChanged(function(v) ESP.Range = v end) -- Color picker local col = espMod:AddColorPicker("Color", Color3.fromRGB(79,142,247)) col:OnChanged(function(c) ESP.Color = c end) -- Notification Nova:Notify("Loaded!", "NovaUI ready.", "success")
